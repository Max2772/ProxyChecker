# ProxyChecker

Асинхронный чекер прокси (HTTP/HTTPS/SOCKS4/SOCKS5) — проверяет список прокси из `proxies.txt` на паре случайных сайтов и выводит рабочие/нерабочие.

## Установка

```bash
uv sync
```

## Использование

Добавь прокси в `proxies.txt` (по одному на строку, с префиксом схемы):

```
socks5://1.2.3.4:1080
http://5.6.7.8:8080
```

Запуск:

```bash
uv run main.py
```

## Известные ограничения

- Результаты выводятся только в консоль, не сохраняются в файл.
- Проверяется только доступность прокси, а не его анонимность.
- Нет ограничения на число одновременных соединений.
